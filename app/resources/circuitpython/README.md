# Bundled CircuitPython UF2

Official CircuitPython build for the Waveshare RP2040-Zero, flashed onto
blank boards by the app's provisioning flow (`provision_flash_uf2`).

- Version: 10.2.1
- Source: https://downloads.circuitpython.org/bin/waveshare_rp2040_zero/en_US/adafruit-circuitpython-waveshare_rp2040_zero-en_US-10.2.1.uf2

The app picks up whatever single `*.uf2` file is in this directory — to
upgrade, replace the file (keep the versioned filename) and update this note.
