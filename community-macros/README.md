# Community macros

Share your MKYADA macros! Open a pull request adding a `.json` file
(format: [docs/macro-format.md](../docs/macro-format.md)) with a short
description in this table.

Import a macro via the app (**Recorder → Import JSON…** or assign it directly
on the **Keys** page), or copy it straight to the board's `macros/` folder as
`key<N>.json`.

| File | What it does | Author |
|---|---|---|
| [inventory-plus-template.json](inventory-plus-template.json) | Template: click an inventory slot, press +, repeat until stopped | @asilbalaban |

> Coordinates in recorded macros are resolution-dependent — check the macro's
> `screen` field matches your display, or re-record with your own layout.
